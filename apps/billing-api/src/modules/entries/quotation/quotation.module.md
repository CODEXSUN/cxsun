# Quotation Entry Module

Quotation Entry is native inside Billing API and owns quotation create, update, suspend, restore, comments, activities, print/PDF/mail handoff, and sales-invoice linkage state.

It keeps the sales repository cross-link native to Billing API so generated sales invoice state can be coordinated without remounting the compatibility quotation module.

## Detailed module note

Read `QUOTATIONENTRY.md` in this folder before changing quotation behavior. It records the current summary, completed scope, known gaps, and future concepts for quotation-to-sales conversion.
