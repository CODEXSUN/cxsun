# Tenant Domain Module

Tenant Domain maps public domains and local hosts to tenants.

## Responsibilities

- Store tenant domain mappings in the master database.
- Resolve request host names to tenant context.
- Mark primary and alias domains.
- Persist resolver settings for landing mode, landing app, industry, and company routing.

## Frontend

Super-admin can manage tenant domains and edit domain settings through visual controls.
