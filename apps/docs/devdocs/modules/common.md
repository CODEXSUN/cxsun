# Common Master Data Module

Common Master Data provides reusable tenant setup records.

## Groups

- Location: countries, states, districts, cities, pincodes.
- Contacts: groups, types, address types, bank names.
- Product: groups, categories, types, units, HSN codes, taxes, brands, colours, sizes, styles.
- Orders: order types, transports, warehouses, destinations, stock rejection types.
- Others: currencies, priorities, payment terms, months, and sales account types.

## Usage

Common records are used by master, entry, stock, and reporting modules.
