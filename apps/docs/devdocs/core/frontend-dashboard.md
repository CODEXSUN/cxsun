# Frontend Dashboard

The frontend dashboard is role-aware and module-driven.

## Super Admin

Super-admin users can access tenant orchestration, tenant domains, industries, app setup, user manager, system update, queue manager, database manager, project docs, and Agent OS admin pages.

## Tenant Workspace

Tenant users see enabled app modules. App availability is controlled by tenant payload settings and dashboard app definitions.

## Navigation

Dashboard navigation is defined in the app sidebar and mapped to page components in the dashboard view. Super-admin-only pages should be added to both the sidebar page type and the super-admin page access list.
