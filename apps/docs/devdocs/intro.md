# Codexsun Developer Docs

Codexsun is a multi-tenant business platform with a platform master database, isolated tenant databases, a React dashboard, backend modules for business operations, and a super-admin operating surface.

This documentation app is the central map for the project. It is maintained as a separate Docusaurus workspace under `apps/docs` and is linked from the super-admin frontend.
