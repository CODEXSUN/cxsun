# Quotation Report

Quotation Report helps review quotation status.

## Use It To

- List all quotations.
- Separate billed and unbilled quotations.
- Review customer quotation totals.
- Follow pending quotes.

## Steps

1. Open Billing > Report > Quotation Report.
2. Select date range and customer if needed.
3. Review billed and unbilled sections.
