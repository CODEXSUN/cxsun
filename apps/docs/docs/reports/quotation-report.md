# Quotation Report

Quotation Report helps review quotation status.

## Use It To

- List all quotations.
- Separate billed and unbilled quotations.
- Review customer quotation totals.
- Follow pending quotes.

## Steps

1. Open Billing > Report > Quotation Report.
2. Select date range and customer if needed.
3. Choose billed, unbilled, or all status.
4. Review billed and unbilled sections.
5. Follow up pending quotations.

## Reading The Report

- Billed quotations already have a linked sale.
- Unbilled quotations still need follow-up.
- Totals help compare quote value by period.
- Customer filter helps review one party at a time.
