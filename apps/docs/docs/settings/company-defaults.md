# Company Defaults

Company defaults control which company and accounting year are active for daily work.

![Company defaults](/img/docs/company-defaults.svg)

## When To Check

- Before entering invoices.
- After switching company.
- At the start of a new accounting year.

## What It Affects

Default company context is used by entries, reports, print formats, and document settings.

## How To Use

1. Open Application > Default Company or the matching settings page.
2. Select company.
3. Select accounting year.
4. Save.
5. Return to Billing and confirm entries show the selected company.

## Important

If company or accounting year is wrong, new entries and reports may appear in the wrong place. Always check this at the start of work.
