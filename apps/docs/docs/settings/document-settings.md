# Document Settings

Document settings control numbering and print behavior for entries.

![Document numbering](/img/docs/document-numbering.svg)

## Typical Settings

- Prefix.
- Starting number.
- Financial year pattern.
- Entry type numbering.
- Print options.

## Where It Is Used

- Sales invoice number.
- Export sales invoice number.
- Purchase entry number.
- Quotation number.
- Receipt and payment numbers.

## Setup Steps

1. Open Settings > Document Settings.
2. Select document type.
3. Set prefix, separator, starting number, and suffix if required.
4. Enable automatic numbering when office users should not type numbers manually.
5. Save and create a test entry before live use.

## Good Practice

Configure document settings before live billing starts. Avoid changing numbering after entries are already created unless your administrator confirms the impact.
