# Workspace Foundation

Codexsun is being organized so each business app can grow separately while still sharing the same secure foundation.

## What The Foundation Handles

- Login and session access
- Tenant and domain identification
- Company and accounting year context
- User roles and permissions
- App enablement
- Shared notifications, mail, files, and audit records as they are introduced

## Why This Matters

Billing, Ecommerce, CRM, Sites, and other apps should feel separate and focused, but users should not have to manage separate logins or repeated setup for each one.

The shared foundation keeps common access and company context consistent across apps.

## Current User Impact

For now, users continue using the existing workspace normally. The backend is being prepared in stages so future deployments can update one app without disturbing another.

No billing records, customers, products, or reports are moved by this preparation step.

## Future Direction

As each app becomes independently deployable:

1. The shared foundation keeps identity, tenant, company, and permission behavior consistent.
2. Each business app owns only its own work.
3. Documentation for user-visible features stays centralized in this guide.
