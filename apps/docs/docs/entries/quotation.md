# Quotation Entry

Use Quotation Entry to prepare a price offer before billing.

## Create A Quotation

1. Open Billing > Quotation.
2. Select the customer.
3. Add quotation date and validity details.
4. Add product or service lines.
5. Review taxes and totals.
6. Save and print the quotation.

## Follow Up

Use Quotation Report to see billed and unbilled quotation details.
