# Quotation Entry

Use Quotation Entry to prepare a price offer before billing. A quotation can later be used for follow-up and converted into a sale where that workflow is enabled.

![Quotation flow](/img/docs/quotation-flow.svg)

## Create A Quotation

1. Open Billing > Quotation.
2. Select the customer.
3. Enter quotation number and date.
4. Add validity or reference details if needed.
5. Add product or service lines.
6. Review tax and total.
7. Save and print or download the quotation.

## When To Use

- Price approval before invoice.
- Customer comparison.
- Follow-up for pending orders.
- Internal approval before supply.

## Follow Up

Use Quotation Report to see billed and unbilled quotation details.
