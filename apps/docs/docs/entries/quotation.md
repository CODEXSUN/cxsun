# Quotation Entry

Use Quotation Entry to prepare a customer price offer before creating a Sales invoice.

![New Quotation entry screen](/img/docs/entries/quotation-entry.png)

## Before You Start

- Create the customer and required addresses.
- Create products with HSN code, unit, GST rate, and price.
- Confirm quotation numbering and the active accounting year.
- Keep work-order or enquiry references ready.

## Create a Quotation

1. Open **Billing > Entries > Quotation**.
2. Select **New Quotation**.
3. Search and select the **Customer name**.
4. Confirm the **Quotation no** and **Date**.
5. Enter the **Work Order no** when applicable.
6. Select **CGST-SGST** or **IGST**.

## Add Quotation Items

1. Search and select the product.
2. Create the product when it does not exist. The new product's HSN code, unit, and GST are populated immediately.
3. Enter PO/DC references when needed.
4. Enter description, quantity, and quoted price.
5. Select **Add**.
6. Repeat for all offered items.
7. Review taxable value, GST, round off, and quotation total.

## Address and Terms

1. Open **Address** and select the required customer address.
2. Create a secondary address when necessary; it is saved and selected immediately.
3. Use **Terms** for validity, delivery, payment, and other quotation conditions.
4. Use E-way or E-invoice tabs only when your workflow requires those details.

## Save, Follow Up, and Convert

1. Select **Save** or **Save & Print**.
2. Send the quotation to the customer by print, PDF, email, or WhatsApp.
3. Track it in **Quotation Report**.
4. When approved, use the invoice-generation action to create a Sales invoice.
5. Verify the generated Sales invoice reference on the quotation.

## Common Mistakes

- Treating a quotation as an accounting invoice.
- Quoting a product without HSN, unit, or GST.
- Omitting validity, delivery, or payment terms.
- Generating another invoice from an already invoiced quotation.
