# Sales Entry

Use Sales Entry to create invoices or sales records for customers.

## Before You Start

- Confirm the active company.
- Create the customer contact if it does not exist.
- Create products with unit, HSN, and tax details.
- Check document numbering settings if invoice numbers are managed automatically.

## Create A Sale

1. Open Billing > Sales.
2. Select or enter the customer.
3. Add invoice date and document details.
4. Add products, quantity, rate, discount, and tax details.
5. Review totals.
6. Save the entry.
7. Print or share the invoice if required.

## Good Practice

Use customer statement reports to review outstanding invoices and receipts.
