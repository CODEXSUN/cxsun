# Dashboard Basics

The dashboard is the main working area. Use the left side menu to move between setup, billing, reports, and settings.

![Dashboard map](/img/docs/dashboard-map.svg)

## Main Areas

- **Application:** company details, default company, accounting year, and users.
- **Masters:** contacts, products, work orders, and reusable records.
- **Billing:** quotation, sales, purchase, receipts, payments, and billing reports.
- **Accounts:** books, ledgers, and account views where enabled.
- **Settings:** document numbering, print settings, and feature switches.

## Navigation

1. Choose an app area from the sidebar.
2. Open the needed page from the menu.
3. Use the breadcrumb/top area to confirm where you are.
4. Use New, Refresh, filters, and row actions from the list toolbar.

## List Pages

Most list pages include:

- Search box.
- Status or date filters.
- Column visibility options.
- Pagination.
- Three-dot row actions for view, edit, suspend, restore, print, or tools.

Search before creating a new record. This avoids duplicate customers, suppliers, products, and document records.

## Entry Pages

Entry pages usually have tabs such as Details, Address, E-way, E-invoice, and Terms. Complete the Details tab first, then use the other tabs only when needed.
