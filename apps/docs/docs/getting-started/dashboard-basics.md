# Dashboard Basics

The dashboard sidebar groups work by application area.

## Common Areas

- Application: company setup, users, default company, and roles.
- Billing: quotation, sales, purchases, receipts, payments, and billing reports.
- Inventory: stock and delivery workflows.
- Settings: document numbering and software settings.

## Navigation

Use the sidebar to open a module. The top bar shows your current area and quick actions such as Home and Logout.

## Search And Filters

Most list pages include search, filters, columns, and pagination. Use them before creating duplicate records.
