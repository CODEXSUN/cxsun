# Tech Pack

Approved default stack:

```text
Language: TypeScript
Runtime: Node.js LTS
Package manager: npm workspaces
Backend HTTP: Fastify behind framework adapter
Backend logging: Pino with pino-pretty in development
Frontend: React + Vite
Frontend styling: Tailwind CSS
Frontend UI base: shadcn/ui compatible primitives
Frontend server state: TanStack Query
Frontend tables: TanStack Table
Database access: Kysely
Database engine: MariaDB
Queue: BullMQ
Queue transport: Redis through ioredis
Mail: Nodemailer
Validation: Zod or equivalent
Testing: Vitest
Browser e2e: Playwright
Process runner: concurrently
Formatting: Prettier
Linting: ESLint
Containers: Docker
```

Any tech pack change requires a decision record.
