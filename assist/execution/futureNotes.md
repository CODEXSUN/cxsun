# Future Notes

## Contact enhancements

These are optional future product features, not current Contact creation bugs:

- **GST details UI:** Allow users to manage multiple GST registrations for one Contact. The underlying data types and persistence already exist, but the Contact form does not currently expose the collection.
- **Merge and deduplication:** Detect duplicate Contacts and safely combine selected records while preserving addresses, communication details, transactions, and references.
- **Bulk import/export:** Create or update Contacts from CSV or Excel, with validation, duplicate handling, error reporting, and an export option.

Implement these as separate planned product slices after the current Contact workflow is stable.
